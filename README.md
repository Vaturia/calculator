# Calculator

## Description
Simple console calculator for UNIX
## How to install
- Clone repository
`git clone https://github.com/Vaturia/calculator.git`
- Install
```bash
~$ cd calculator
~$ cmake -B build
~$ cmake --build build
~$ sudo cmake --build build --target install
```
- For get help use:
```bash
~$ calculator --help
#or
~$ calculator -h
```
